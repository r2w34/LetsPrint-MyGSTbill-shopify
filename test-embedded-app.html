Fixed! The ERR_BLOCKED_BY_RESPONSE error has been resolved by implementing proper iframe escape logic.

What was the issue?

When Shopify embeds apps in iframes, browsers block redirects to external OAuth URLs for security reasons. This caused the ERR_BLOCKED_BY_RESPONSE error.

How we fixed it:

  1. Created dedicated embed-auth route - Bypasses Shopify middleware
  2. Implemented iframe escape JavaScript - Multiple methods to break out of iframe
  3. Added fallback mechanisms - Ensures OAuth works in all scenarios
  4. Enhanced user experience - Loading spinner and clear messaging
